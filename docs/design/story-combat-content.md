# 剧情战斗内容设计文档

## 概述

本文档定义剧情系统相关的战斗关卡、副本、道具、装备等内容，使剧情中的战斗节点能够真正触发战斗。

## 剧情章节与战斗对应关系

### 序章：机缘初现
- 无战斗内容（纯剧情引导）

### 第二章：拜入仙门
- **实战比试**：与青云宗弟子切磋
  - 敌人：青云宗考核弟子（练气期）
  - 难度：入门级
  - 奖励：青云袍、灵石x50

### 第三章：外门历练
- **妖狼巢穴**：调查村民失踪，击败妖狼
  - 敌人：小妖狼x2 + 妖狼王（Boss）
  - 难度：中等
  - 奖励：聚元丹、清风剑、灵石x100

### 第四章：魔影初现（规划）
- **魔修袭击**：抵御魔修入侵
  - 敌人：魔修弟子x3
  - 难度：困难
  - 奖励：辟邪符、玄铁甲

## 战斗节点集成

剧情中的 `battle` 类型节点需要关联对应的副本ID：

```typescript
// 示例：第三章妖狼王战斗节点
{
  id: 'chapter_3_boss_battle',
  type: 'battle',
  battleId: 'story_3_2',  // 关联副本ID
  next: 'chapter_3_victory'
}
```
